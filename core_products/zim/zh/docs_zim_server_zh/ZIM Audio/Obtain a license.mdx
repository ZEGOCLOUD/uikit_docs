# Obtain a license

- - -

## Description

You can call this method to obtain the license of the audio component.

## Request method and endpoint

- Request method: GET
- Request endpoint: https://auth-api.zego.im?Action=CgiDescribeZIMAudioLicense
- Transmission protocol: HTTPS
- QPS limit: 20 calls per second


## Request parameters

The following table describes only some common request parameters. For the complete list of common request parameters, see [Accessing Server APIs](../Accessing%20Server%20APIs.mdx#public-parameters).

## Sample request

```json
https://auth-api.zego.im?Action=CgiDescribeZIMAudioLicense
&<Common request parameters>
```
