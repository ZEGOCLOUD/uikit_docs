# Modify group specification limits

- - -

## Description
You can call this method to modify the following group settings:
- Group joining mode
- Inviter mode
- Invitee mode
- The maximum number of group members


After the modification is successful, group members can receive notifications of the group specification limit changes through the following ZIM SDK callback interface:

## Request method and endpoint

- Request method: GET
- Request endpoint: https://zim-api.zego.im/?Action=ModifyGroupLimit
- Protocol: HTTPS
- Rate limit: 20 request/second


## Request parameters

The following table describes only the method-specific request parameters and some common request parameters. For the complete list of common request parameters, see [Accessing Server APIs - Public parameters](../Accessing%20Server%20APIs.mdx#public-parameters).

<th>Type</th>

<tr data-row-level="2">
<td>FromUserId</td>
<td>String</td>
<td>Yes</td>
<td>Modify the operator's user ID (registered).</td>
</tr>
<tr data-row-level="3">
<td>GroupId</td>
<td>String</td>
<td>Yes</td>
<td>The ID of the target group.</td>
</tr>
<tr data-row-level="4">
<td>JoinMode</td>
<td>Number</td>
<td>No</td>
<td>Mode for joining the group:
<ul><li>0: (Default value) No verification is required, anyone can join the group directly.</li><li>1: The user needs to initiate an application to join the group, and the user can join the group only after the group owner or group administrator approves it.</li><li>2: External users must be invited by members of the group to join the group.</li></ul></td>
</tr>
<tr data-row-level="5">
<td>InviteMode</td>
<td>Number</td>
<td>No</td>
<td>Invitation mode:
<ul><li>0: (Default value) All group members can invite external users to the group.</li><li>1: Only the group owner or group administrator can initiate invitations.</li></ul></td>
</tr>
<tr data-row-level="6">
<td>BeInvitedMode</td>
<td>Number</td>
<td>No</td>
<td>Invited mode:
<ul><li>0: (Default value) External users can be directly invited to the group without the other party's consent. </li><li>1: After inviting someone to join the group, the other party’s consent is required. </li></ul></td>
</tr>
<tr data-row-level="7">
<td>MemberCountLimit</td>
<td>Number</td>
<td>No</td>
<td>The maximum number of group members, the default value is 0.</td>
</tr>
</tbody></table>

<Note title="Note">

  - The FromUserID and GroupId parameters can contain only digits, letters, and the following characters:{" '!'，'#'，'$'，'%'，'&'，'('，')'，'+'，''，':'，';'，'<'，'='，'.'，'>'，'?'，'@'，'['，']'，'^'，'_'，'{'，'}'，'|'，'~'"}.
  - If there is no need to modify `JoinMode`, `InviteMode` or `BeInvitedMode`, please do not carry this parameter in the request URL. If a parameter is carried but no value is assigned, the value of the parameter may be modified to 0.

</Note>


## Sample request

```json
https://zim-api.zego.im/?Action=ModifyGroupLimit
&<Common request parameters>
&FromUserId=zego
&GroupId=group
&JoinMode=1
&MemberCountLimit=100
```
