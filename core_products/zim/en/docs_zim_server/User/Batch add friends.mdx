# Batch add friends

- - -

## Overview

You can call this operation to batch add up to 20 friends for a user without their consent. If you need a higher limit (50 at maximum), contact ZEGOCLOUD technical support.

Each user can have a maximum of 3000 friends.

## Operation prototype

- Request method: POST
- Request URL: `https://zim-api.zego.im/?Action=AddFriends`
- Protocol: HTTPS
- QPS limit: 20 calls per second

## Sample request

- Request URL:
  
  ```json
  https://zim-api.zego.im/?Action=AddFriends
  &<Common request parameters>
  ```

- Request body:

```json
{
  "FromUserId": "zego_user",
  "FriendInfos": [
      {
          "UserId": "usera1",
          "Wording": "Wording1",
          "FriendAlias": "alias1",
          "Attributes": [
              {
                  "Key": "k1",
                  "Value": "Value"
              },
              {
                  "key": "k2",
                  "Value": "Value1"
              }
          ]
      },
      {
          "UserId": "usera2",
          "Wording": "Wording2",
          "FriendAlias": "alias2",
          "Attributes": [
              {
                  "key": "k3",
                  "Value": "Value"
              },
              {
                  "key": "k0",
                  "Value": "Value1"
              }
          ]
      }
  ]
}
```
