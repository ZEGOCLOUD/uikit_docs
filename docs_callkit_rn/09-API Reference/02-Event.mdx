# Invitation Events



## onIncomingCallDeclineButtonPressed

>
> This callback will be triggered to **callee** when callee click decline button in incoming call
>
>- function prototype:
>
>```js
>onIncomingCallDeclineButtonPressed(navigation)
>```
>
>- example:
>
>```js
>return ZegoUIKitPrebuiltCallService.init(
>  Your_APPID,
>  Your_APPSign,
>  Your_UserID,
>  Your_UserName,
>  [ZIM, ZPNs],
>  {
>    onIncomingCallDeclineButtonPressed: (navigation) => {
>      //...
>    },
>  }
>);
>```

## onHangUp

> user will receive this callback when all other persons leaves call.usually you can end call in this callback.
>
>- function prototype:
>```js
>onHangUp(duration)
>```
>- example:
>```js
>return ZegoUIKitPrebuiltCallService.init(
>  Your_APPID,
>  Your_APPSign,
>  Your_UserID,
>  Your_UserName,
>  [ZIM, ZPNs],
>  {
>    requireConfig: (data) => {
>      return {
>        onHangUp: (duration) => {
>          //...
>          props.navigation.navigate('HomeScreen');
>        }
>      }
>    }
>  }
>);
>```
